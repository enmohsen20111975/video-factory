## 1. البدء السريع

### تشغيل النظام

```bash
# Terminal 1: Backend
npm run dashboard

# Terminal 2: Frontend
cd dashboard-app && npm run dev

# Terminal 3: Queue Worker
npm run queue-worker
```

افتح المتصفح على: `http://localhost:3000`

### الصفحة الرئيسية

ستجد:
- **📊 إحصائيات**: عدد الكتب، الدروس، الفيديوهات
- **📚 آخر الكتب**: قائمة بأحدث الكتب
- **🎬 قائمة الانتظار**: الفيديوهات قيد التوليد

## 2. رفع الكتب

### الخطوات

1. اذهب لصفحة **📚 Books** من الـ sidebar
2. اضغط زر **"+ رفع كتاب جديد"**
3. املأ النموذج:
   - **العنوان**: اسم الكتاب (مثل "الفيزياء للصف الثالث الثانوي")
   - **المادة**: اختر من القائمة (physics, chemistry, ...)
   - **الصف**: اختر المرحلة الدراسية
   - **الناشر**: اسم الناشر (اختياري)
   - **ملف PDF**: ارفع ملف PDF الممسوح
4. اضغط **"رفع"**

### ملاحظات

- ✅ يدعم PDF حتى 100MB
- ✅ عدد الصفحات يُحسب تلقائياً
- ✅ يمكن رفع عدة كتب في نفس الوقت

---

## 3. استخراج المحتوى

### بدء الاستخراج

1. اضغط على الكتاب في القائمة
2. في صفحة تفاصيل الكتاب، اضغط **"🚀 بدء الاستخراج"**
3. سيظهر شريط التقدم

### ماذا يحدث أثناء الاستخراج؟

1. **المرحلة 1** (5%): تحويل PDF لصور (150 DPI)
2. **المرحلة 2** (10-80%): استخراج المحتوى بالـ VLM
   - صفحة بصفحة (10 ثواني لكل صفحة)
   - استخدام qwen2.5vl:7b
3. **المرحلة 3** (80-90%): دمج الصفحات
4. **المرحلة 4** (90-100%): توليد master.json + lesson.json

### مراقبة التقدم

- شريط التقدم في صفحة الكتاب
- السجل اللحظي في صفحة Video Studio
- حالة النظام في الـ Header

### إيقاف الاستخراج

- اضغط **"⏸ إيقاف"** في أي وقت
- يمكن الاستئناف لاحقاً (يستأنف من حيث توقف)

## 4. مراجعة وتعديل الدروس

### فتح محرر الدرس

1. اذهب لصفحة الكتاب
2. في شجرة الدروس، اضغط على أي درس
3. سيفتح محرر الدرس مع التبويبات

### التبويبات

#### 📝 النص (Content)
- **النص الخام**: النص المستخرج من الكتاب
- **الملخص**: ملخص الدرس
- **الأهداف**: قائمة بأهداف الدرس
- **التعريفات**: المصطلحات وتعريفاتها
- **الشرح**: شرح تفصيلي مع الصور

#### 🖼️ الصور (Images)
- عرض جميع الصور المستخرجة
- **إضافة صورة**: اضغط "+ رفع صورة"
- **حذف صورة**: اضغط زر الحذف على الصورة
- **تعديل الوصف**: اضغط على وصف الصورة لتعديله

#### 📊 الجداول (Tables)
- عرض الجداول المستخرجة
- **تعديل خلية**: اضغط على الخلية
- **إضافة صف/عمود**: استخدم الأزرار
- **حذف صف/عمود**: استخدم أزرار الحذف

#### 🧮 الصيغ (Formulas)
- عرض الصيغ بصيغة LaTeX
- **معاينة فورية**: KaTeX renders الصيغة
- **تعديل LaTeX**: اضغط على الصيغة
- **المتغيرات**: عدّل الرموز والوحدات

#### ❓ الأسئلة (Questions)
- 4 أنواع:
  - **MCQ**: اختيار من متعدد
  - **Numerical**: إجابة رقمية
  - **Conceptual**: إجابة نصية
  - **True/False**: صح/خطأ
- **إضافة سؤال**: اضغط "+ سؤال جديد"
- **تعديل**: اضغط على أي جزء من السؤال

#### 🎬 الفيديو (Video)
- **السكريبت**: النص الصوتي (يُولد تلقائياً)
- **الصوت**: اختر صوت TTS
- **توليد الفيديو**: اضغط الزر
- **معاينة**: شغل الفيديو بعد التوليد

### الحفظ

- **حفظ تلقائي** كل 30 ثانية
- **حفظ يدوي**: اضغط **"💾 حفظ"**
- **مؤشر التغييرات**: يظهر بجانب العنوان

### تمييز كمراجَع

بعد مراجعة الدرس:
1. اضغط **"✅ تمييز كمراجَع"**
2. أضف ملاحظات (اختياري)
3. الآن الدرس جاهز لتوليد الفيديو

## 5. توليد الفيديوهات

### توليد فيديو لدرس واحد

1. افتح الدرس في المحرر
2. اذهب لتبويب **🎬 الفيديو**
3. تأكد من السكريبت (أو عدّله)
4. اختر الصوت
5. اضغط **"🎬 توليد الفيديو"**

### توليد فيديوهات متعددة

1. اذهب لـ **🎬 Video Studio**
2. اختر الكتاب من القائمة المنسدلة
3. حدد الدروس (checkboxes)
4. اضغط **"🎬 توليد المحدد"**

### مراقبة التوليد

- **الحالة**: في صفحة Video Studio
- **السجل اللحظي**: يظهر في الأسفل
- **التقدم**: لكل درس على حدة

### ماذا يحدث أثناء التوليد؟

1. **توليد السكريبت** (إذا لم يوجد)
2. **توليد الصوت** (Edge-TTS)
3. **توليد التوقيت** (Word boundaries)
4. **Remotion Render** (إنتاج الفريمات)
5. **FFmpeg Compress** (ضغط MP4)
6. **تحديث lesson.json** (بالنتيجة)

المدة المتوقعة: 3-5 دقائق لكل درس (75 ثانية فيديو)

### إلغاء التوليد

- اضغط **"❌ إلغاء"** في صفحة Video Studio
- سيتوقف بعد الخطوة الحالية

### تحميل الفيديو

- اضغط **"⬇️ تحميل"** بجانب الفيديو المكتمل
- أو شغّله مباشرة في المتصفح

## 6. التصدير لمنصة education

### الخطوات

1. اذهب لصفحة الكتاب
2. اضغط **"📤 تصدير لـ education"**
3. سيتم توليد ملف `education-export.json`
4. حمّل الملف

### تنسيق الملف

```json
{
  "exported_at": "2025-01-15T10:00:00Z",
  "book": {
    "id": "physics-3rd-secondary",
    "title": "الفيزياء للصف الثالث الثانوي",
    "subject": "physics",
    "grade": "3rd-secondary"
  },
  "lessons": [
    {
      "lesson_id": "lesson-1-1",
      "title": "قانون أوم",
      "summary": "...",
      "video_url": "/data/books/.../videos/lesson-1-1.mp4",
      "duration_sec": 75,
      "questions": [...],
      "formulas": [...]
    }
  ]
}
```

### الرفع على منصة education

1. ارفع ملف `education-export.json` على استضافة education
2. في إعدادات المنصة، حدد مسار الملف
3. ستظهر الدروس والفيديوهات تلقائياً

## 7. الإعدادات المتقدمة

### الوصول للإعدادات

اذهب لـ **⚙️ Settings** من الـ sidebar

### إعدادات VLM

| الإعداد | الافتراضي | الوصف |
|---------|----------|-------|
| النموذج المفضل | qwen2.5vl:7b | أفضل للعربي والصيغ |
| Cooldown | 10 ثواني | بين كل صفحة |
| VRAM Limit | 7GB | منع OOM |
| Temperature | 0.1 | دقة الاستخراج |

### إعدادات TTS

| الإعداد | الافتراضي | الوصف |
|---------|----------|-------|
| الصوت | ar-EG-SalmaNeural | صوت مصري |
| السرعة | +5% | أسرع قليلاً |
| النبرة | +0Hz | طبيعي |

**أصوات متاحة**:
- `ar-EG-SalmaNeural` - سلمى (مصر)
- `ar-EG-ShakirNeural` - شاكر (مصر)
- `ar-SA-NorahNeural` - نورة (سعودية)
- `ar-AE-FatimaNeural` - فاطمة (إمارات)

### إعدادات الفيديو

| الإعداد | الافتراضي | الوصف |
|---------|----------|-------|
| FPS | 30 | إطار/ثانية |
| الدقة | 1920×1080 | Full HD |
| Concurrency | 4 | أنوية المعالج |
| CRF | 22 | جودة الضغط |
| Preset | fast | سرعة الضغط |

### إعدادات Cloudflare R2 (اختياري)

لرفع الفيديوهات تلقائياً على R2:

1. أنشئ حساب على [Cloudflare R2](https://www.cloudflare.com/products/r2/)
2. أنشئ bucket جديد
3. احصل على API tokens
4. أدخل في الإعدادات:
   - Account ID
   - Access Key ID
   - Secret Access Key
   - Bucket Name
   - Public URL Base
5. فعّل "Auto Upload"
6. اضغط **"اختبار الاتصال"**

## 8. الأسئلة الشائعة

### س: كم يستغرق استخراج كتاب 200 صفحة؟
**ج**: حوالي 3-4 ساعات (10 ثواني/صفحة + معالجة)

### س: كم حجم الفيديو الواحد؟
**ج**: 5-15 MB لفيديو 75 ثانية (1920×1080, H.264)

### س: هل يمكنني استخدام الكتاب بدون VLM؟
**ج**: نعم، يمكنك إنشاء دروس يدوياً من صفحة الكتاب

### س: ماذا لو فشل استخراج صفحة؟
**ج**: يتم تسجيل الخطأ ومتابعة باقي الصفحات. يمكنك إعادة المحاولة لاحقاً

### س: هل يمكن تعديل السكريبت الصوتي؟
**ج**: نعم، في تبويب "🎬 الفيديو" في محرر الدرس

### س: كيف أضيف صوتاً مختلفاً؟
**ج**: في الإعدادات > TTS، أو لكل درس على حدة

### س: هل يدعم النظام اللغة الإنجليزية؟
**ج**: حالياً مصمم للعربية. يمكن إضافة دعم الإنجليزية مستقبلاً

### س: ما الحد الأقصى لحجم PDF؟
**ج**: 100MB (قابل للتعديل في dashboard-server.js)

### س: هل يمكنني تشغيل النظام على خادم سحابي؟
**ج**: نعم، لكن يحتاج GPU للـ VLM. استخدم خوادم مثل RunPod أو Lambda Labs

### س: كيف أنشئ نسخة احتياطية؟
**ج**: انسخ مجلد `data/` بالكامل
